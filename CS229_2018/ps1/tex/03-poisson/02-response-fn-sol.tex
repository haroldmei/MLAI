\begin{answer}

The canonical response function for Poisson is just:
\begin{eqnarray*}
\mathbb{E}(y|\eta) 
    &=& \lambda \\
    &=& \exp(\eta)
\end{eqnarray*}
where $\eta = \theta^T x$ is the natural parameter. The last step above uses 
previous derived result for the natural parameter: $\eta = \log \lambda$.

\end{answer}
